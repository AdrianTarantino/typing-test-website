# typing-test-website
Full-stack app for typing test. Inspired by MonkeyType.

## Requirements
- Basics
    - Random words appear on screen.
    - User can type those words. Correct letters turn green incorrect letters turn red, Excess letter are grey (similar to monkeytype).
    - Users can login using Google or GitHub account. Accounts store average typing time and best typing time.
        - Use Firebase for storing all account related stuff.
    - Users can select between 15 seconds, 30 seconds and 1 minute.
